## 🎯 核心功能实现

### 已实现
1. **报价管理**
   - 创建、查询、更新、删除报价记录
   - 支持按件号搜索
   - 数据状态管理（有效/无效/待确认）

2. **比价功能**
   - 按件号聚合多家供应商报价
   - 自动计算并高亮最低人民币单价
   - 美金单价和人民币单价分列显示

3. **汇率换算**
   - 可配置的换算公式：人民币单价 = 美金单价 × 汇率 + 附加费用 + (美金单价 × 服务费率)
   - 业务人员可在系统设置界面调整参数
   - 历史数据保留当时的公式参数快照

4. **邮件数据提取框架**
   - HTML 表格解析
   - 非结构化文本提取
   - PDF 附件解析（pdfplumber）
   - 图片 OCR 框架（pytesseract）
   - "暂无报价"自动识别

5. **可扩展架构**
   - 数据库抽象层（SQLite → PostgreSQL 无需改业务代码）
   - 用户模型预留（user_id 字段）
   - 多用户扩展准备

## 📝 关键技术决策

1. **技术栈选择**
   - 后端：Python + FastAPI（快速开发、丰富的数据处理库）
   - 前端：React + TypeScript + Ant Design（现代化、类型安全）
   - 数据库：SQLite（阶段一）→ PostgreSQL（阶段二）

2. **架构设计**
   - 前后端分离：独立开发、部署
   - 数据库抽象层：使用 SQLAlchemy ORM，便于数据库切换
   - 服务层模式：业务逻辑与 API 分离

3. **汇率换算设计**
   - 公式可配置，业务人员可调整
   - 历史数据快照，保证可追溯性
   - 对比人民币单价，符合实际业务需求
